高中体育批量评分系统：Sport Score System
==============

## 简介
高中体育批量评分系统是一个专为中国高中学生体育测试成绩进行批量评分的工具，目前该系统拥有 `Web` 、`Windows` 、`Linux (Debain/Ubuntu)` 三种平台版本，后续还会逐步增加对 `MacOS` 、`Android` 、`HarmonyOS` 等平台的支持。
